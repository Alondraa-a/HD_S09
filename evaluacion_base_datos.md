# Evaluación técnica de base de datos

Durante el desarrollo se detectó que la base de datos actual no soporta algunas funcionalidades clave.

## Acciones
- Evaluar si conviene migrar a otra base de datos.
- Analizar si se puede complementar la solución actual.
- Documentar riesgos técnicos.
- Revisar impacto en tiempo y costos.

## Riesgos
- Retraso en el desarrollo.
- Mayor costo de implementación.
- Posibles cambios en la estructura del sistema.

## Decisión inicial
Se realizará una evaluación técnica antes de decidir una migración completa.
